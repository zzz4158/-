### 相关开发框架：
> * Umi (https://umijs.org/zh/)
> * Dva (https://dvajs.com/)
> * React (https://react.docschina.org/)
> * Redux（https://www.redux.org.cn/)

### 组件和动效：
> * Ant Design (https://ant.design/)
> * Ant Motion (http://motion.ant.design/)
> * APlayer (https://aplayer.js.org/)
